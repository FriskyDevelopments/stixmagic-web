Telegram VideoChat Companion Bot

A practical bot for Telegram communities that want basic room automation, live session announcements, speaker queue helpers, and clear operational status during voice or video chats.

Current capabilities

  • Detects Telegram video chat start and end service events.
  • Tracks invited participants from Telegram service messages.
  • Provides /vc, /status, /join, and /leave commands.
  • Keeps an in-memory speaker queue for lightweight room moderation.

Who it is for

  • Telegram group admins running recurring voice rooms.
  • Community moderators managing speaker order manually.
  • Teams that want a clean starting point instead of a demo project.

Known limits

Telegram bots do not expose every live room signal. This repo only claims support for the service events and commands that are actually implemented today. Deeper participant presence tracking remains a documented next step.
